README

内容： I2C通信

目的： 
1. 理解I2C主/从设备是如何工作的；
2. 理解I2C基本协议的内容（使用协议分析仪测量并分析I2C信号）；

要求：
1. 准备两块Arduino UNO主板，其中一块为Maser device, 另外一块为Slave device;
2. 分别为这两块Arduion UNO主板写入代码，并实现通过I2C通信；
3. 完成并提交代码；
4. 完成作品后，拍照并提交；

思考：
1. 如果不用Arduino的I2C库函数，如何实现I2C的操作？
2. 如果只使用普通的GPIO pin脚（不支持I2C协议），如何使用软件模拟I2C协议？
3. 两块Arduino如何通过UART进行通信？

参考资料：
1. Arduino相关的I2C主/从设备代码，请到网上自行查找；
2. I2C基本协议内容，建议参考 郭天祥《51单片机C语言编程》，
   8.1节 “I2C总线概述”
   8.2节 “单片机模拟I2C总线通信”
